# Contributing to awtSCADA

Thank you for your interest in contributing to awtSCADA!

## How to Contribute

### Reporting Bugs
- Use [GitHub Issues](https://github.com/larionovavi-stack/bitscada/issues)
- Include browser version, OS, and steps to reproduce

### Feature Requests
- Open a [Discussion](https://github.com/larionovavi-stack/bitscada/discussions)
- Describe the use case and expected behavior

### Protocol Requests
We welcome requests for additional industrial protocols:
- IEC 60870-5-101/104
- DNP3
- BACnet
- Profinet
- EtherNet/IP

### Code Contributions
The full source code is available under commercial license.
For the demo version, you can contribute:
- Documentation improvements
- Bug reports
- Translation to other languages
- Graphic element designs (SVG)
